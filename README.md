**QUIZ GAME** เป็น game ที่จะทำการสุ่มคำถามที่มีให้ทั้งหมด10คำถามโดยจะมีตัวเลือกให้เลือกทั้งหมด 4 ข้อในการเล่นผู้เล่นก็จะต้องทำการอ่านคำถามแล้วก็ทำการตอบให้ครบตามจำนวนข้อแล้วคะแนนก็จะออกมาแสดงให้เห็น
